## 3. app/core/graph.py - DIJKSTRA SHORTEST PATH
**Railway rerouting when tracks blocked**

```python
from typing import Dict, List, Tuple
import heapq  # Priority queue for Dijkstra

class Graph:
    def __init__(self) -> None:
        self.adj: Dict[str, List[Tuple[str, float]]] = {}  # Adjacency list

    def add_edge(self, u: str, v: str, w: float) -> None:
        self.adj.setdefault(u, []).append((v, w))  # Add u->v weight w
        self.adj.setdefault(v, []).append((u, w))  # Undirected: v->u also

    def remove_edge(self, u: str, v: str) -> None:
        # Remove blocked edges (construction/accident)
        if u in self.adj:
            self.adj[u] = [(node, w) for node, w in self.adj[u] if node != v]
        if v in self.adj:
            self.adj[v] = [(node, w) for node, w in self.adj[v] if node != u]

    def dijkstra(self, source: str, target: str) -> Tuple[float, List[str]]:
        dist: Dict[str, float] = {node: float("inf") for node in self.adj}
        prev: Dict[str, str | None] = {node: None for node in self.adj}
        dist[source] = 0.0  # Source distance 0

        heap: List[Tuple[float, str]] = [(0.0, source)]  # Min-heap (distance, node)

        while heap:
            d, u = heapq.heappop(heap)  # Get closest unvisited
            if u == target: break
            if d > dist[u]: continue  # Stale entry
            for v, w in self.adj.get(u, []):  # Neighbors
                nd = d + w  # New distance
                if nd < dist[v]:  # Better path
                    dist[v] = nd
                    prev[v] = u
                    heapq.heappush(heap, (nd, v))  # Update heap

        # Reconstruct path using prev pointers
        if dist[target] == float("inf"):
            return float("inf"), []
        path = []
        cur = target
        while cur is not None:
            path.append(cur)
            cur = prev.get(cur)
        path.reverse()
        return dist[target], path
```
**DSA Learning**: Classic Dijkstra with binary heap. Time O((V+E)logV). Handles edge removal for real-time rerouting.

## 4. app/core/platform_engine.py - MIN PLATFORMS GREEDY
```python
from typing import List, Tuple

def min_platforms(intervals: List[Tuple[int, int]]) -> int:
    \"\"\"
    intervals: (arrival, departure) minutes
    \"\"\"
    events = []
    for arr, dep in intervals:
        events.append((arr, 1))  # Arrival +1 platform
        events.append((dep, -1)) # Departure -1 platform

    events.sort()  # Sweep line by time
    current, max_platforms = 0, 0
    for time, delta in events:  # Process events chronologically
        current += delta
        max_platforms = max(max_platforms, current)
    return max_platforms
```
**DSA Learning**: Sweep line algorithm. Sort all events, scan once O(n log n). Perfect for overlapping intervals.

**Example**: [[900,910],[940,1200],[950,1120],[1100,1130]] → 3 platforms

## 5. app/core/coach_dll.py - FAULTY COACH LOCALIZATION
**Doubly Linked List for O(n) first fault find**

```python
class CoachNode:
    def __init__(self, coach_id: str, faulty: bool = False):
        self.coach_id = coach_id
        self.faulty = faulty
        self.prev = None
        self.next = None

class CoachDLL:
    def __init__(self): self.head = self.tail = None

    def append(self, coach_id: str, faulty: bool = False):
        node = CoachNode(coach_id, faulty)
        if not self.tail:
            self.head = self.tail = node
        else:
            self.tail.next = node
            node.prev = self.tail
            self.tail = node  # O(1) tail append

    def to_list(self) -> List[str]:
        res = []
        cur = self.head
        while cur:
            res.append(cur.coach_id)
            cur = cur.next  # O(n) traversal
        return res

    def find_faulty_index(self) -> int:
        idx = 0
        cur = self.head
        while cur:
            if cur.faulty: return idx  # First fault O(n) single pass
            cur = cur.next
            idx += 1
        return -1  # No fault
```
**DSA Learning**: DLL advantages: tail append O(1), bidirectional. Here simulates train coaches sequence.

## 6. app/core/safety_controller.py - EMERGENCY SAFETY FSM
```python
from enum import Enum

class TrainState(str, Enum):
    MOVING = "MOVING"
    STOPPED = "STOPPED"
    EMERGENCY_REQUESTED = "EMERGENCY_REQUESTED"

class SafetyController:
    def __init__(self, unsafe_zones: List[Tuple[int, int]]):
        self.state = TrainState.MOVING
        self.unsafe_zones = unsafe_zones  # Bridges/tunnels

    def is_in_unsafe_zone(self, position: int) -> bool:
        return any(start <= position <= end for start, end in self.unsafe_zones)

    def nearest_safe_position(self, position: int) -> int:
        for start, end in self.unsafe_zones:
            if start <= position <= end:
                # Nearest exit
                return start - 1 if (position - start) <= (end - position) else end + 1
        return position

    def handle_emergency(self, position: int) -> Tuple[TrainState, int]:
        self.state = TrainState.EMERGENCY_REQUESTED
        if self.is_in_unsafe_zone(position):
            safe_pos = self.nearest_safe_position(position)
            self.state = TrainState.STOPPED
            return self.state, safe_pos  # Move to safety first
        self.state = TrainState.STOPPED
        return self.state, position
```
**Design Pattern**: Finite State Machine (FSM). Real-world safety: stop in unsafe zone → nearest safe km.

## 8. ROUTERS (app/routers/*.py) - API Endpoints
**Clean separation**

**network.py** example:
```python
from fastapi import APIRouter
router = APIRouter()  # Sub-router

@router.post("/shortest-path", response_model=PathResponse)  # POST + response model
def shortest_path(req: NetworkRequest):  # Auto validates input
    graph = Graph()
    for edge in req.edges: graph.add_edge(edge.u, edge.v, edge.w)
    for blocked in req.blocked_edges: graph.remove_edge(blocked.u, blocked.v)
    distance, path = graph.dijkstra(req.source, req.target)
    return PathResponse(distance=distance, path=path)
```
**Pattern**: Router per feature → main.py includes with prefix/tags.

## HOW TO BUILD FROM SCRATCH

1. **Setup**: `mkdir rail_intelligence && cd rail_intelligence`
2. **Dependencies**: Create requirements.txt, `pip install -r requirements.txt`
3. **Structure**: Create directories `app/core`, `app/models`, etc.
4. **Core DSA**: Copy core/*.py - understand each algorithm
5. **Models**: Define Pydantic schemas matching cores
6. **Routers**: Wire cores to POST endpoints
7. **Main**: App + CORS + router includes
8. **Frontend**: HTML forms calling APIs
9. **Test**: Unit tests + manual Swagger
10. **Run**: `uvicorn app.main:app --reload`

**Learning path**: DSA → FastAPI → Frontend integration → Production CORS/security.

**Production tips**: Add auth, Docker, PostgreSQL for persistent graphs, WebSocket real-time trains.
